# AIタスク司令塔

Slackに届いた依頼文をコピペするだけで、AIが **タスク分解・優先度スコア・工数見積もり・スケジュール配置・返答テンプレ** をまるごと自動生成するローカルWebアプリ。

---

## 趣旨

事業会社のエンジニア（インフラ〜アプリ横断）は、Slackで「ちょっとこれやっといて」系の依頼が日常的に飛んでくる。
しかしその場で「何日かかるか」を根拠付きで即答するのは難しく、見積もりを後回しにするとタスクが積み上がって優先度が曖昧になる。

**このツールは「なんで3日かかるの？」に根拠付きで即答できるようにする**ための業務効率化ツール。

- 依頼文をそのままコピペ → AIが自動でタスク分解・工数・優先度・スケジュールを出力
- 返答テンプレが自動生成されるので、Slackにそのまま貼り付けるだけ
- バッファ（レビュー待ち・環境不安定など）を追加すると返答テンプレを再生成
- `PROJECT.md` でプロジェクト固有情報（デプロイルール・チーム構成など）を注入すると精度が上がる

---

## クイックスタート

```bash
pip3 install -r requirements.txt
python3 app.py
```

ブラウザで `http://localhost:5000` を開く。
