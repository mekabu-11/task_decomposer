# Task Decomposer

Slackに届いた依頼文をコピペするだけで、AIが **工数見積もり・作業手順・Backlogチケット・Slack返信文** をまるごと自動生成するローカルWebアプリ。

---

## 趣旨

事業会社のエンジニア（インフラ〜アプリ横断）は、Slackで「ちょっとこれやっといて」系の依頼が日常的に飛んでくる。
しかしその場で「何日かかるか」を根拠付きで即答するのは難しく、見積もりを後回しにするとタスクが積み上がる。

**このツールは「なんで3日かかるの？」に根拠付きで即答できるようにする**ための業務効率化ツール。

- 依頼文をそのままコピペ → AIが工数・作業手順・Backlogチケット・Slack返信を自動生成
- 返答テンプレが自動生成されるので、Slackにそのまま貼り付けるだけ
- バッファ（レビュー待ち・環境不安定など）をAI分解前に設定可能
- コンテキストファイルでプロジェクト固有情報を注入すると精度が上がる

---

## クイックスタート

```bash
pip3 install -r requirements.txt
python3 app.py
```

ブラウザで `http://localhost:5001` を開く。
